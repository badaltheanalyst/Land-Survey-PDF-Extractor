# Land Survey PDF Extractor

## Overview

A Python automation tool that extracts land survey numbers from PDF documents and exports clean, structured Excel reports.

This project was developed to automate a repetitive manual process and improve data accuracy when working with large survey-related PDF files.

---

## Business Problem

Land survey documents often contain hundreds of survey numbers embedded within PDF files.

Manually identifying, extracting, cleaning, and organizing these numbers into Excel can be time-consuming and prone to human error.

---

## Solution

This tool automates the entire workflow:

* Reads PDF documents
* Extracts survey numbers using Regex patterns
* Cleans and standardizes number formats
* Removes duplicates
* Sorts records
* Exports results to Excel
